Networking:

For the client's network, I recommend using a wireless wifi router for simplicity and good price. I would recommend using a client server instead of peer to peer, because it has a central server where users can access most files, and that is what the clients want (to collaborate). Clients would use Git and Github, and I would recommend Atom (the real time text editor). I would recommend peer to mail method for file sharing, since has no proplems with blocking ports, has high speed tranfer, etc.
